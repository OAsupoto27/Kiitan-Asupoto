# Event Booking System

## Project Description
A full-stack web application that allows users to browse and book events online.

## Features
- User registration and login
- Event creation
- Event booking
- Booking cancellation
- Admin dashboard

## Technologies Used
- Next.js
- React
- Node.js
- MongoDB

## Installation

1. Clone the repository

2. Install dependencies
npm install

3. Start the development server
npm run dev

4. Open in browser
http://localhost:3000
